## 🔧 本地访问方式

### 方式1：直接打开HTML文件
```bash
# 在浏览器中打开
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```

### 方式2：使用Python简单HTTP服务器
```bash
# 进入项目目录
cd /path/to/deploy

# 启动HTTP服务器（端口8000）
python3 -m http.server 8000

# 然后在浏览器访问：
# http://localhost:8000
```

- **技术博客**：查看`index.html`和`articles.html`

## 🚀 在线部署建议

### GitHub Pages（推荐）
1. 创建GitHub仓库
2. 将`deploy/`目录内容推送到仓库
3. 启用GitHub Pages功能
4. 访问：`https://[用户名].github.io/[仓库名]/`

### Vercel（全栈部署）
1. 注册Vercel账号
2. 导入GitHub仓库
3. 自动部署静态网站
4. 免费SSL和CDN

### Netlify（简单部署）
1. 拖拽`deploy/`文件夹到Netlify
2. 自动生成网址
3. 支持自定义域名
